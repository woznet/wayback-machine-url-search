Wayback Machine URL Search
==========================

The "Wayback Machine URL Search" is a Google Chrome extension that provides a way to search for a given URL on the [Internet Archive Wayback Machine](https://web.archive.org/) directly from the browser.

Features
--------

- **Context Menu:** Right-click on a link or selected text to search it on the Wayback Machine.
- **Toolbar Popup:** Click the extension icon to search the current page on the Wayback Machine.
- **Settings:** Toggle the toolbar popup on or off from the extension options page.

Installation
------------

To install the extension, follow these steps:

1.  Clone or download the repository.
2.  Open Google Chrome and go to the "Extensions" page (chrome://extensions/).
3.  Enable "Developer mode" by toggling the switch in the top-right corner.
4.  Click on the "Load unpacked" button in the top-left corner.
5.  Navigate to the repository folder and select the `src` directory.

Usage
-----

### Context Menu

1.  Right-click on a link or select text on a webpage.
2.  Click on the "Wayback Machine URL Search" menu item in the context menu.
3.  The Wayback Machine search page will open in a new tab with the selected URL.

### Toolbar Popup

1.  Click the Wayback Machine URL Search icon in the Chrome toolbar.
2.  Click the "Search Current Page" button.
3.  The Wayback Machine search page will open for the current page's URL.
